# Документация API модуля погоды

## Основные функции

### index(city_name)
Получает текущую погоду для указанного города.

**Параметры:**
- `city_name` (строка): Название города для поиска погоды.

**Возвращает:**
Словарь с информацией о текущей погоде:
- `city`: Название города
- `description`: Описание погоды
- `icon`: Код иконки погоды
- `temperature`: Строка с текущей температурой в °C
- `country_code`: Код страны
- `wind`: Скорость ветра в км/ч
- `humidity`: Влажность в %
- `time`: Текущее время и дата

**Пример:**
```python
weather_info = index("Москва")
print(weather_info["temperature"])  # Выводит "Температура: 15 °C"
```

### get_month_forecast(city_name)
Получает прогноз погоды на несколько дней вперед.

**Параметры:**
- `city_name` (строка): Название города для поиска погоды.

**Возвращает:**
Словарь с прогнозом погоды:
- `forecasts`: Список строк с прогнозом на каждый день
- Или `error`: Строка с описанием ошибки, если запрос не выполнен

**Пример:**
```python
forecast = get_month_forecast("Москва")
if "forecasts" in forecast:
    for day_forecast in forecast["forecasts"]:
        print(day_forecast)  # Выводит "25 октября: 10°C, переменная облачность"
```

### get_full_forecast(city_name)
Объединяет текущий прогноз и прогноз на месяц.

**Параметры:**
- `city_name` (строка): Название города для поиска погоды.

**Возвращает:**
Словарь с двумя ключами:
- `day`: Результат выполнения функции `index()`
- `month`: Результат выполнения функции `get_month_forecast()`

**Пример:**
```python
full_forecast = get_full_forecast("Москва")
print(full_forecast["day"]["temperature"])
print(full_forecast["month"]["forecasts"][0])
```

### get_details_forecast(city_name)
Получает детальный прогноз на утро, день и вечер.

**Параметры:**
- `city_name` (строка): Название города для поиска погоды.

**Возвращает:**
Словарь с прогнозом на части дня:
- `morning`: Прогноз на утро
- `day`: Прогноз на день
- `evening`: Прогноз на вечер
- Или `error`: Строка с описанием ошибки, если запрос не выполнен

**Пример:**
```python
details = get_details_forecast("Москва")
print(f"Утро: {details['morning']}")
print(f"День: {details['day']}")
print(f"Вечер: {details['evening']}")
```

### get_weather_by_coords(lat, lon)
Получает погоду по географическим координатам.

**Параметры:**
- `lat` (число): Широта
- `lon` (число): Долгота

**Возвращает:**
Словарь с информацией о погоде (аналогичен возвращаемому значению функции `index()`)
или словарь с ключом `error`, содержащим описание ошибки.

**Пример:**
```python
weather = get_weather_by_coords(55.7558, 37.6173)  # Москва
print(weather["temperature"])
```

### get_coordinates(city_name)
Получает географические координаты города по его названию.

**Параметры:**
- `city_name` (строка): Название города для поиска координат.

**Возвращает:**
Словарь с координатами:
- `lat`: Широта
- `lon`: Долгота
- Или `error`: Строка с описанием ошибки, если город не найден

**Пример:**
```python
coords = get_coordinates("Москва")
if "error" not in coords:
    print(f"Широта: {coords['lat']}, Долгота: {coords['lon']}")
```
